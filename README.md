# NATIONAL-DEBT

Статичний Telegram-friendly сайт для GitHub Pages у стилі боргового табло.

## Файли

- `index.html` - розмітка застосунку.
- `styles.css` - темне LED-табло в стилі debt clock.
- `app.js` - дані боргів, рендер і кнопки поширення.

## Де міняти борги

Відкрий `app.js` і зміни масив `debts`.

## GitHub Pages

У репозиторії відкрий `Settings -> Pages`, вибери деплой з гілки `main` і папку `/ (root)`.

Для Telegram Mini App URL буде виглядати приблизно так:

```text
https://<username>.github.io/NATIONAL-DEBT/
```
